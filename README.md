# 🌍 지구를 지켜라! 환경 퀴즈 웹사이트

## 프로젝트 개요
초등학생을 대상으로 한 환경 교육 퀴즈 웹사이트입니다.  
100가지 환경 관련 퀴즈를 통해 지구 환경을 생각하고 고민할 수 있도록 설계되었습니다.

---

## ✅ 구현된 기능

### 퀴즈 기능
- **100문항 환경 퀴즈** (8개 카테고리)
- **카테고리 선택**: 전체 / 기후&날씨 / 재활용&분리수거 / 동물&생태계 / 바다&물 / 에너지&자원 / 숲&식물 / 환경오염
- **문제 수 선택**: 10 / 20 / 50 / 100문제
- **선택지 랜덤 셔플** (매 게임마다 선택지 순서 변경)
- **즉각적인 정답/오답 피드백** (색상, 아이콘, 애니메이션)
- **해설 표시** (각 문제마다 교육적 설명 제공)

### 점수 & 결과
- **실시간 점수 집계** (정답당 10점)
- **점수 카운트업 애니메이션**
- **5단계 등급 시스템**: 환경 박사 → 환경 지킴이 → 환경 탐험가 → 환경 새싹 → 환경 씨앗
- **정답률, 정답 수, 오답 수 통계**
- **오답 복습 기능** (틀린 문제 + 정답 + 해설 표시)

### UI/UX (초등학생 친화적)
- **큰 글씨와 이모지** 활용
- **바운싱 지구 애니메이션**
- **+10점 팝업 효과**
- **정답 시 펄스 애니메이션 / 오답 시 흔들기 애니메이션**
- **진행 바(Progress Bar)** 표시
- **완전 반응형 디자인** (모바일·태블릿·PC 지원)
- **Google Fonts(Noto Sans KR)** 적용

## 🎮 이용 방법

1. `index.html` 접속
2. 카테고리와 문제 수 선택
3. **퀴즈 시작하기** 버튼 클릭
4. 4개 보기 중 정답 선택
5. 해설 확인 후 다음 문제로 이동
6. 퀴즈 완료 후 점수 및 등급 확인
7. 오답 복습 또는 다시 도전
